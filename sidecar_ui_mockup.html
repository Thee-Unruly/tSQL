Which products are low on stock and haven't been restocked in the last 30 days?
Generated SQL Copy code
SELECT p.product_name, i.quantity, i.last_restocked
FROM product_master p
JOIN inventory i ON p.id = i.product_id
WHERE i.quantity < 10
  AND i.last_restocked < NOW() - INTERVAL '30 days'
ORDER BY i.quantity ASC;
